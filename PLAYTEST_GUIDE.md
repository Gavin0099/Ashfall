# Ashfall Playtest 引導指南 (PT-1)

感謝你參與 《Ashfall》 的核心機制測試。本階段我們的目標是驗證遊戲的「決策張力」。

## 🚀 如何開始測試

1.  **啟動遊戲**：
    執行以下指令進入測試模式：
    ```bash
    python scripts/play_cli_run.py 101 --playtest
    ```
    *(註：`101` 是指定的測試種子，`--playtest` 會啟動後台計時器。)*

2.  **遊戲目標**：
    你的唯一目標是：**活到終點山脊 (Final Ridge)**。

---

## 📋 觀察者指南 (Observer Only)

如果您是本次測試的觀察者，請在旁紀錄以下行為：

- **猶豫點**：玩家在選擇路徑或事件選項時，是否停頓超過 5 秒？
- **情緒反應**：玩家是否發出「糟糕」、「死定了」或嘆氣等情緒信號？
- **混亂點**：玩家是否對某個數值變化表示不解？

請使用 [playtests/observation_sheet_template.md](file:///e:/BackUp/Git_EE/Ashfall/playtests/observation_sheet_template.md) 紀錄。

---

## 🎤 結束後訪談 (Interview)

請在玩家完成（或死亡）後，詢問以下 3 個核心問題：

1. **最艱難的抉擇**：剛才哪一個路徑或事件讓你最難下決定？
2. **死因/危急判斷**：當你死亡（或生命瀕危）時，你覺得是什麼原因造成的？（是剛才的選擇，還是純粹運氣？）
3. **悔恨與改變**：如果讓你重跑一次，你會在什麼地方做不同的選擇？

---

## 💾 數據收集

測試結束後，系統會在 `output/playtests/` 生成一個 JSON 檔案。請將該檔案連同您的觀察紀錄回傳給開發團隊。

> [!NOTE]
> 我們關心的是您的「決策過程」而非您的「生存技術」。請盡情按照您的直覺去玩。

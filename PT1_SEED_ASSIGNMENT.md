# PT-1 Seed Assignment

## Purpose

Use a fixed seed table so the first Blind CLI round has controlled coverage.

This is not for balance proof.
It is for structured human observation.

| Player | Seed | Expected Run Family Bias | Notes |
|---|---:|---|---|
| P1 | 101 | north_aggressive | strong early pressure |
| P2 | 103 | south_aggressive | food relief + radiation tension |
| P3 | 104 | south_cautious | lower explicit pressure, good for contrast |
| P4 | 105 | mixed_pressure | mixed route pressure |
| P5 | 1001 | exploratory | check whether non-baseline seed still reads clearly |

## Minimal Round

If only 3 players are available:

1. `101`
2. `103`
3. `105`

## Why Fixed Seeds

- easier machine-vs-human comparison
- easier session-to-session discussion
- avoids confusing early feedback with seed noise

## After PT-1

If the first round is useful, PT-2 can widen seed coverage.
